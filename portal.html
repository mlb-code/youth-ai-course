❓ עזרה ושאלות

מדריכי התקנה ותשובות לשאלות נפוצות

איך מתקינים Cursor? +
  1. היכנסו לאתר cursor.com
  2. לחצו על "Download" — האתר יזהה אוטומטית אם אתם במק או ווינדוס
  3. הריצו את קובץ ההתקנה שירד
  4. במק: גררו את האייקון לתיקיית Applications
  5. בווינדוס: לחצו Next עד שההתקנה מסתיימת
  6. פתחו את Cursor ותירשמו עם חשבון Google או אימייל
איך מתקינים Node.js? +
  1. היכנסו ל-nodejs.org
  2. הורידו את גרסת LTS (הגרסה היציבה)
  3. הריצו את קובץ ההתקנה
  4. לחצו Next בכל השלבים (אל תשנו הגדרות)
  5. לבדיקה: פתחו Terminal (מק) או CMD (ווינדוס) והקלידו node --version
  6. אם מופיע מספר גרסה — ההתקנה הצליחה!
איך פותחים פרויקט חדש ב-Cursor? +
  1. פתחו את Cursor
  2. לחצו File → Open Folder
  3. צרו תיקייה חדשה בשם הפרויקט שלכם (בלי רווחים!)
  4. בחרו את התיקייה ולחצו Open
  5. לחצו Ctrl+I (ווינדוס) או Cmd+I (מק) לפתיחת צ'אט AI
  6. כתבו מה אתם רוצים לבנות ו-Cursor יעזור!
מה לעשות אם המחשב איטי? +
  • סגרו תוכנות שלא צריכים (במיוחד דפדפנים עם הרבה טאבים)
  • הפעילו מחדש את המחשב לפני המפגש
  • ודאו שיש לפחות 4GB RAM פנויים
  • אם Cursor עדיין איטי: כתבו לנו בוואטסאפ ונעזור!
איך נרשמים ל-ChatGPT / Claude? +

ChatGPT:

  1. היכנסו ל-chat.openai.com
  2. לחצו "Sign up" ותירשמו עם Google או אימייל
  3. הגרסה החינמית מספיקה לקורס!

Claude:

  1. היכנסו ל-claude.ai
  2. לחצו "Sign up" ותירשמו עם Google או אימייל
  3. הגרסה החינמית מספיקה לקורס!
איך מתחברים למפגש בזום? +
  1. לפני כל מפגש תקבלו לינק בוואטסאפ
  2. לחצו על הלינק — זום ייפתח אוטומטית
  3. אם זום לא מותקן: הורידו מ-zoom.us/download
  4. ודאו שהמיקרופון והמצלמה עובדים
  5. היו מוכנים 5 דקות לפני!
